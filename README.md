# Название проекта
Этот проект представляет собой электронный магазин, который я создаю для своего портфолио. На данный момент имеются 
несколько страниц: главная, о нас, профиль, корзина, страницы регистрации и логина. Если пользователь в группе seller, он может
добавлять товары через специальную страницу. Любой пользователь может добавить товары в корзину и оплатить через подключенную
в тестовом режиме юкассу.

## Использование
для использования проекта скопируйте репозиторий на сервер
```sh
git clone https://github.com/thelight21313/mini-shop
```
затем соберите контейнер 

```sh
sudo docker-compose up --build -d
```
контейнер должен запуститься сам, но если этого не произошло пропиште 

```sh
sudo docker-compose up
```
